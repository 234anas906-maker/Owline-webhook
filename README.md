* **Purpose:** Public-facing dashboard & documentation.
* **License:** `MIT`
* **Path:** `./packages/website`

### 📦 [02] Components SDK
* **Purpose:** Core engine for Discord V2 components.
* **License:** `PolyForm Noncommercial`
* **Path:** `./packages/components-sdk`

### 🛠️ [03] Quick Commands
* **Initialize:** `yarn install`
* **Build SDK:** `cd components-sdk && yarn build`
* **Run Dashboard:** `cd website && yarn dev`
